Welcome!

Thank you for visiting "Lost In Translation".

This site demonstrates how language translation (especially computer-based translation) is challenging and at times funny.

Example

Using google translate, start with the following English phrase:
"You can lead a horse to water, but you can't make him drink"
Translate it into Chinese:
你可以带马去喝水，但你不能让他喝酒
Taking this translation and translating it back to english yields:
"You can take a horse to drink water, but you can not let him drink".

Try "Echo" Translation

To try a phrase for yourself, click on "Try".

This takes you to a tool where enter you are prompted to enter an english phrase. The tool will translate it into chinese. Then, it will translate that back into english as an "echo". Often, it is very accurate. But, sometimes, something gets "lost in translation".
